# Hướng dẫn cài đặt

## Bước 1: Cài đặt yêu cầu
1. Đảm bảo bạn đã cài đặt Git và Node.js trên máy.
2. Cài đặt các gói cần thiết bằng command:
   ```bash
   npm install
   ```

## Bước 2: Chạy ứng dụng
1. Sử dụng lệnh sau để khởi động ứng dụng:
   ```bash
   npm start
   ```
